== Description ==

Autoplay music with cookie is a wordpress plugin that enable music autoplay in your wordpress website. The plugin has cookie support so you can enable when music play time like once an hour or once a day.

You can upload various music type via option panel & change settings.

<h2>Main Features</h2>

<ul>
<li>Lightweight - only 4 KB!</li>
<li>Easy installation.</li>
<li>jQuery cookie support!</li>
<li>Cross-browser support!</li>
<li>conditional music.</li>
</ul>

See live demo here: http://wpexpand.com/plugins/premium-plugins/autoplay-music/

== Installation ==

Installing this plugin as regular wordpress plugin. 

After install, you can see a menu, settings > Autoplay music. Upload your music there.
